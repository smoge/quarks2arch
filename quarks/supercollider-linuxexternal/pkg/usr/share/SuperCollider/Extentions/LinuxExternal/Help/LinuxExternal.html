LinuxExternal


This Quark bundles a couple of classes created to interface external programs on Linux


SCJMeterBridge
A Class to automatically start and connect the JACK Meterbridge to the SC outputs.
You need the program "meterbridge" for this to work.


SCJConnection
A Class to manage JACK connections from SuperCollider.
You need the jack tools, "jack_lsp", "jack_connect" and "jack_disconnect" for this to work.


GNUPlot
A Class to use plot functions from GNUPlot from SuperCollider.
You need to have installed "gnuplot" for this to work.


RecordMyDesktop
A Class to create a screencast of your SuperCollider session.
You need to have installed "recordmydesktop" for this to work.
